DarkflameServer Web Dashboard

A modern C++ web interface for managing your Darkflame Universe server.

Features
  • Account Management: Create, modify, ban, lock, and mute player accounts
  • Character Management: View, rescue, and manage player characters
  • Moderation Tools: Approve pet names, manage properties, and review bug reports
  • Mail System: Send in-game mail to players with item attachments
  • Play Keys: Manage registration keys for new accounts
  • Activity Logs: Monitor player activity and track logins/logouts
  • Audit Trail: Track all administrative actions for accountability
Technology Stack
  • Backend: C++ with Crow web framework
  • Frontend: Bootstrap 5, jQuery, DataTables
  • Templates: Mustache templating engine
  • Database: MySQL/MariaDB or SQLite

About Darkflame Universe

DarkflameServer is an open-source server emulator for LEGO Universe, a massively multiplayer online game that was officially discontinued in 2012. The Darkflame Universe project aims to preserve and revive this beloved game for fans to continue enjoying.
